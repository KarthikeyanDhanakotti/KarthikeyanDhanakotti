# Hi, I'm Karthikeyan Dhanakotti 👋

**Enterprise AI Architect** · Independent open-source work

I build **open architectures**, **reference implementations**, and **research** for production-grade Enterprise AI and Agentic AI systems.

My work centres on **Enterprise AI Systems Engineering** — turning what today are one-off AI stacks into an engineered discipline with a shared vocabulary, verifiable outputs, honest cost models, and standards-aligned governance.

## Current Focus

- 🏗 **Enterprise AI Systems Reference Architecture (EASRA)** — an open architecture standard for Enterprise AI, akin in intent to TOGAF or the OSI model.
- ⚙ **Enterprise AI Execution Compiler (EAEC)** — research: compile agent workflows to constraint-verified execution graphs. *(in EASRA/research)*
- 🔍 **Enterprise AI Verification Framework (EAVF)** — research: verification as a first-class layer, distinct from evaluation. *(in EASRA/research)*
- 🤖 Agentic AI & Multi-Agent Systems.
- 🛡 Enterprise AI Security (OWASP LLM, MITRE ATLAS).
- 📊 AI Observability & LLMOps.

## Open Source

- ⭐ **[EASRA](https://github.com/KarthikeyanDhanakotti/EASRA)** — Vendor-neutral open architecture standard for Enterprise AI. Twelve specs, five architecture views, four cloud mappings (Azure / AWS / GCP / open-source), a security + verification + LLMOps reference, benchmarks scaffold, and a diagram catalogue.
